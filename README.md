# Cafe-Sales-Dashboard
Cafe Sales Dashboard | Excel

This project showcases an interactive Cafe Sales Dashboard built using Microsoft Excel to analyze sales performance and customer behavior.

Data Preparation

Cleaned and transformed raw data using Power Query.

Created calculated columns for Day Name and Shift.

Applied basic data modeling to ensure accurate analysis.

 Key Insights

Total Sales: 42.6K

Total Orders: 1,313

Top Product: Americano With Milk

Peak Shift: Morning

Most Used Payment Method: Card (93%)

Tools Used

Microsoft Excel

Power Query

Data Modeling

This dashboard helps identify sales trends, popular products, and peak operating times for better decision-making.
